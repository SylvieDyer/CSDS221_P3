Welcome to Minesweeper!

If you're unfamiliar with the rules of this game, below is a brief explanation:

Your goal is to uncover all of the mines properly. At the top of the board, you can see how many mines there are hidden in the board.

If you suspect a mine is hidden beneath the tile, right click the tile to flag it- you win when you have flagged all of the hidden mines.

If you suspect no mines hide beneath a tile, click it to reveal how many mines it touches- you lose if you select a bomb.
A tile can touch anywhere between 0 and 8 mines as indicated by the number on the tile once it is revealed. (This includes all of the tile's edges, along with each of its corners)

Finally, you can only place as many flags as there are bombs, so place wisely! If you place all of your allotted flags, but do not successfully identify all of the bombs, a message will appear.

If you're up for a challenge or want to start off with fewer tiles, you can even change the board width with the width picker, and hit set board width to adjust the width!

Begin the game by clicking any tile- hopefully it's not a bomb!
Have fun!
